# /verify:audit

Read the `verify` skill and execute the `/verify:audit` workflow.

**Steps**:
1. Scope the surface to audit (phase, sprint, feature, or file set)
2. Deploy parallel explorers per perspective (see the `patterns` skill's multi-explorer pattern)
3. Consolidate findings; each explorer returns a bounded summary

See **SKILL.md § /verify:audit** for full details.
