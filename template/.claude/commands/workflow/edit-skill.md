# /workflow:edit-skill

Read the `workflow` skill and execute the `/workflow:edit-skill` workflow.

**Steps**:
1. Identify the system artifact to change (a skill, hook, rule, or `CLAUDE.md`) and read it in full
2. Make the edit, preserving the existing shape and conventions of that artifact
3. Verify the change (dry-run or regenerate as applicable) and record it in the state chain
